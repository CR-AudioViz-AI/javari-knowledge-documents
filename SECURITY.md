# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please send an email to security@craudiovizai.com.

Do not open public issues for security vulnerabilities.

## Security Measures

- All data is encrypted at rest
- Access is controlled via authentication
- Regular security audits are performed
- Dependency vulnerabilities are monitored

## Contact

For security concerns: security@craudiovizai.com
For general inquiries: support@craudiovizai.com
