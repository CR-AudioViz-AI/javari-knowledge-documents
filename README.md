# Javari Knowledge Documents

Knowledge base documents for the Javari AI assistant.

## Overview

This repository contains curated knowledge documents that power Javari AI contextual understanding across various domains.

## Structure

```
/documents
  /business      - Business and entrepreneurship knowledge
  /creative      - Creative tools and design principles
  /technical     - Technical documentation and guides
  /community     - Community guidelines and social impact
```

## Usage

Documents are processed by the Javari Knowledge Pipeline and indexed for semantic search.

## Contributing

1. Add documents in Markdown format
2. Follow the naming convention: `category-topic-version.md`
3. Include metadata header with tags and description
